# Data Analytics & Insights: Excel Portfolio

This repository showcases a series of data analysis projects developed during the Data Technician program. These projects demonstrate a high level of proficiency in Microsoft Excel, covering the full data lifecycle—from cleaning and transformation to complex modeling and executive visualization.

## 🚀 Key Technical Proficiency
* **Data Transformation:** Text-to-Columns, Trimming, Data Type conversion, and `SWITCH` logic.
* **Advanced Analytics:** Pivot Tables (Grouping, Slicers, Calculated Fields).
* **Logical Functions:** `VLOOKUP`, `SUMIFS`, `MAX`, `AVERAGE`, and Nested IF statements.
* **Visualization:** Dynamic Line Charts, Stacked Column Charts, and Demographic Pie Charts.
* **Compliance & Ethics:** Implementation of GDPR and Data Protection Act (2018) principles within data handling.

---

## 📂 Project Case Studies

### 1. Global Bike Sales Dashboard
A comprehensive end-to-end analysis of a global retail dataset (2017–2021) to identify growth drivers and profit margins.

* **Process:** Aggregated over 5 years of sales data using Pivot Tables.
* **Key Insight:** Identified "Adults (35-64)" as the most profitable demographic, contributing over $47M in revenue.
* **Visualization:** Created a multi-chart dashboard to compare annual performance against demographic distribution.

### 2. Retail Operations & Automation (`SWITCH` Logic)
This project focused on automating the categorization of high-volume retail data to improve operational efficiency.

* **Process:** Cleaned raw transaction data using the "Text-to-Columns" feature to fix formatting errors.
* **Logic:** Implemented the `SWITCH` function to automatically bucket sales volumes into "Low," "Medium," and "High" categories for inventory reporting.

### 3. HR & Payroll Modeling
A simulated Human Resources environment used to manage departmental budgets, headcount, and payroll calculations.

* **Process:** Built a payroll engine that calculates Net Pay based on dynamic rules (Tax: 32%, SS: 8%, Pension: 5%).
* **Analysis:** Used Pivot Tables to segment headcount and total salary spend by Gender and Department (Sales, Purchasing, Logistics).
* **Impact:** Developed a "Summary Sheet" that provides an at-a-glance view of organizational costs.
